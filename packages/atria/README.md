## What is atria?

`atria` is the CLI package for the Atria ecosystem.

Atria is a document-first CMS in active development, based on a custom grammar for structured content and editorial workflows.

This project is developed as part of the Master in Graphic Design and Editorial Projects, Faculty of Fine Arts in Porto.

Atria project is built from a design perspective:

- Built by designers, for designers.
- Content treated as an editorial system.
- Structured workflows with creative flexibility.
- Clear technical tooling and predictable runtime behavior.

## Project Status

Atria is in an experimental phase and evolves rapidly.

- APIs and command behavior may change.
- Some features are still under development.

## Getting Started

Create a new project:

```bash
npm create atria@latest
```

Run the development server:

```bash
cd your-project
npm run dev
```
